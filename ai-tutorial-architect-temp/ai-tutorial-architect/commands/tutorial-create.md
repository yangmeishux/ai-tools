# /tutorial:create - Create Complete Tutorial

Launch the full tutorial creation workflow with all 10 agents.

## Usage

```
/tutorial:create [topic] [--level=beginner|intermediate|advanced] [--time=short|medium|full]
```

## Examples

```
/tutorial:create "React Hooks" --level=beginner --time=full
/tutorial:create "Python Async" --level=intermediate --time=medium
```

## What It Does

1. Chief coordinates the full workflow
2. All 10 agents participate
3. Quality gates at each phase
4. Gold standard output

## Time Estimates

| Level | Time | Output |
|-------|------|--------|
| short | 2-3h | Quick start guide |
| medium | 4-5h | Standard tutorial |
| full | 6-8h | Comprehensive tutorial |
