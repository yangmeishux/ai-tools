# 最佳实践模板

> 适用于行业规范、团队标准、经验总结类内容
> 质量目标：🥇 金牌标准（专业、规范、实战验证）
> Agent 配置：Chief → Researcher → Practitioner → Writer → Fact-Checker → Editor → Chief

---

## 🎭 Agent 协作流程

```
[Chief] 明确实践范围和应用场景
    ↓
[@researcher] 调研行业标准、收集社区经验、分析权威指南
    ↓
[@practitioner] 验证实践方案的可行性、测试代码示例 ⚠️ 关键环节
    ↓
[@writer] 撰写最佳实践规范、编写示例代码
    ↓
[@fact-checker] 验证技术准确性、检查规范合理性
    ↓
[@editor] 规范格式、统一术语、优化表达
    ↓
[Chief] 审核并定稿
```

### 不适用场景

- 📐 **规范性**：明确的标准和要求

- ✅ **实战验证**：经过真实项目检验
- ⚠️ **陷阱提示**：标注常见错误和反模式
